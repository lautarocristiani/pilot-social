# Historias de Usuario

Este documento detalla los requerimientos funcionales del proyecto **PilotSocial** desde la perspectiva del usuario final.

## US-01: Listado de Publicaciones (Feed)
- **Como** usuario visitante,
- **Quiero** visualizar una lista de todos los posteos disponibles,
- **Para** explorar el contenido de la plataforma rápidamente.

## US-02: Búsqueda y Filtrado
- **Como** usuario explorador,
- **Quiero** poder buscar posts por palabras clave o filtrarlos por autor,
- **Para** encontrar información específica sin hacer scroll infinito.

## US-03: Detalle de Publicación
- **Como** lector interesado,
- **Quiero** acceder a una vista detallada al hacer clic en un post,
- **Para** leer el contenido completo y ver la discusión asociada.

## US-04: Gestión de Comentarios
- **Como** usuario participativo,
- **Quiero** poder agregar comentarios en las publicaciones,
- **Para** compartir mi opinión.
- **Nota:** Mis comentarios deben permanecer visibles incluso si recargo la página (persistencia local).

## US-05: Personalización (UI)
- **Como** usuario,
- **Quiero** poder alternar entre modo claro y oscuro,
- **Para** ajustar la interfaz a mi preferencia visual.
