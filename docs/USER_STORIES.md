# Historias de Usuario

Este documento detalla los requerimientos funcionales del proyecto **PilotSocial** desde la perspectiva del usuario final.

## US-01: Listado de Publicaciones (Feed)
- **Como** usuario visitante,
- **Quiero** visualizar una lista de todos los posteos disponibles en la plataforma,
- **Para** poder explorar el contenido y seleccionar temas de mi interés.

## US-02: Detalle de Publicación
- **Como** lector interesado,
- **Quiero** acceder a una vista detallada de un post específico,
- **Para** leer el contenido completo y visualizar los comentarios asociados al mismo.

## US-03: Creación de Comentarios
- **Como** usuario participativo,
- **Quiero** tener la posibilidad de redactar y enviar un nuevo comentario en un post,
- **Para** compartir mi opinión o feedback sobre el tema tratado.
